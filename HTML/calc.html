A Really Stupid Calculator

A calculator too high effort too ever be useful

You must push only binary literals to the stack and use the operators at the side

Command Function
> Push literal to stack
; End current literal
< Pop from stack
+ Add top two items on stack
- Subtract top two items on stack
* Multiply top two items on stack
/ Divide top two items on stack
% Perform modulo on the top item of the stack and following literal
^ Perform exponent on the top item of the stack using the following literal
